# KARTHA

KARTHA é um jogo de cartas estratégico desenvolvido no GameMaker.

O jogador invoca criaturas em um campo de batalha dividido em trilhas (lanes), gerencia sua mão de cartas e enfrenta inimigos utilizando mecânicas inspiradas em jogos de cartas táticos e RPGs de mesa.

## Principais Mecânicas

- Sistema de cartas colecionáveis
- Campo de batalha com 3 trilhas
- Movimento automático das tropas até a linha de combate
- Combate baseado em rolagem de dados
- Dano, defesa física e defesa mágica
- Sistema de turnos
- IA para oponente
- Sacrifício de criaturas para invocação de cartas mais fortes

## Sistema de Combate

Cada criatura possui:

- Vida
- Dado de dano
- Modificador de dano
- Defesa física
- Defesa mágica

Durante o combate é realizada uma rolagem de acerto utilizando um D20.

- 1: Falha crítica e contra-ataque
- 2–10: Erro
- 11–19: Acerto normal
- 20: Acerto crítico
